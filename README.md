【游戏名称】台球游戏 - 移动端优化版

【游戏介绍】
这是一款H5原生开发的体育类小游戏，现已全面优化移动端体验。游戏采用Canvas 2D渲染引擎开发，提供流畅的台球游戏体验。玩家需要控制台球杆，击打白球，将彩色球击入球袋中。游戏具有真实的物理引擎，模拟真实的台球碰撞和运动轨迹。

【移动端新特性】
✅ 完整的触摸控制系统
✅ 智能屏幕适配（支持横屏/竖屏）
✅ 虚拟控制按钮优化
✅ 高DPI屏幕支持
✅ 防误触设计
✅ 流畅的触摸响应
✅ 自动方向检测和提示

【玩法说明】
移动端操作：
- 🎱 直接触摸击球：点击屏幕任意位置，白球会朝该方向击球
- ⚡ 力度按钮：调整击球力度（可选）
- 🎱 智能击球：自动瞄准最近的球进行击球
- 🔄 重新开始：重置游戏状态
- 📱 建议横屏游戏以获得最佳体验

桌面端操作：
- 鼠标左键按住调整击球力度，松开左键击球
- 鼠标移动控制瞄准方向

游戏规则：
- 使用白球击打彩色球
- 将彩色球击入球袋中
- 每次击球后，球会自然停止
- 将所有彩色球击入袋中即可获胜
- 不能将白球击入袋中，否则失败
